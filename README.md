# RETRO ALARM - Android App

Ретро-будильник для Android с математическими головоломками.

## Функции

- Ретро-стиль в стиле LCD дисплеев 80-х
- Три уровня сложности математических примеров:
  - **EASY**: сложение, вычитание, умножение (1-9)
  - **MEDIUM**: более сложные примеры (10-50)
  - **HARD**: составные примеры (100+100)
- Мелодия будильника и вибрация
- Кнопка SNOOZE (отложить на 5 минут)
- Сохранение будильника после перезагрузки

## Сборка

1. Откройте проект в Android Studio
2. Дождитесь синхронизации Gradle
3. Запустите на эмуляторе или устройстве

Или через командную строку:
```bash
cd RetroAlarm
./gradlew assembleDebug
```

APK будет в `app/build/outputs/apk/debug/app-debug.apk`

## Установка

1. Скопируйте APK на устройство
2. Разрешите установку из неизвестных источников
3. Установите приложение
4. Откройте и установите будильник
